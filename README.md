# Resume Automation Platform

An AI-powered platform to optimize resumes for ATS compatibility using LaTeX.

## Quick Start

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)

## Features

- User authentication (login/signup)
- Resume configuration management
- AI-powered resume optimization
- LaTeX-based resume generation
- PDF preview and download

## Tech Stack

- Next.js 16.1.6
- React 19.2.3
- TypeScript
- Tailwind CSS
- Zustand
- Axios
